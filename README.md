# 재고관리 시스템

Next.js로 만든 간단한 재고관리 프로그램입니다.

## 기능

- ✅ 제품 등록 (제품명, 사진, 구입가, 판매가, 링크)
- ✅ 마진율 자동 계산
- ✅ 제품 수정 및 삭제
- ✅ 로컬스토리지를 사용한 데이터 저장
- ✅ 반응형 디자인 (웹, 모바일 모두 지원)
- ✅ 다크모드 지원

## 시작하기

### 1. 의존성 설치

\`\`\`bash
npm install
\`\`\`

### 2. 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

### 3. 빌드 및 배포

\`\`\`bash
npm run build
npm start
\`\`\`

## 기술 스택

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Data Storage**: LocalStorage (브라우저)

## 주요 기능 설명

### 제품 등록
- 제품명, 사진, 구입가, 판매가, 링크 입력
- 마진율은 자동으로 계산됨: `(판매가 - 구입가) / 구입가 × 100`

### 제품 관리
- 등록된 제품 목록 확인
- 수정 버튼으로 제품 정보 수정
- 삭제 버튼으로 제품 삭제

### 데이터 저장
- 브라우저의 LocalStorage에 자동 저장
- 페이지 새로고침 후에도 데이터 유지

## 모바일 앱 사용

이 웹앱은 PWA(Progressive Web App)로 변환 가능하며, 모바일 브라우저에서도 최적화되어 있습니다.

모바일에서 "홈 화면에 추가"를 통해 앱처럼 사용할 수 있습니다.
